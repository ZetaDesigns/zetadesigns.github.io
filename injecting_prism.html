Injecting Pokemon Prism Save

This guide was written by Rid1 and can also be found on Reddit

What you will need

  • Your latest patched Pokemon Prism save file and your installed Pokemon Prism cia
  • A Hex Editor (preferably HxD, noob friendly)

Section I - Creating a fresh save

  1. If you havent already play for a few minutes up to the point where you fight your first Shinx, after that you can save.
    • Save In Game, NOT using the VC Save State function
  2. Exit out of the game and open up JKSM
  3. In the JKSM menu press (A) on SD/CIA
  4. Hit (A) on Pokemon Prism
  5. Hit (A) on Save Data Options
  6. Hit (A) on Export Save
  7. Hit (A) on new and enter the name Fresh and hit (Start)
    • A message box saying "Complete!" should appear if succesful, if not repeat steps 3a-3e It may not prompt you to hit new if you don't have any previously exported saves for Prism
  8. Exit out of JKSM by hitting (B) a few times and power of your 3ds
  9. Insert your SD Card into your PC

Section II - Injecting the save

  1. Download the latest release of the Pokemon Prism Save Converter
  2. Extract the .7z file to somewhere easily accesible e.g Desktop
  3. Navigate to X:\SD\JKSV\Saves\Pokemon_Prism\Fresh, right click on the sav.dat and press Copy
  4. Open the Pokemon Prism Save Converter folder and right click on empty space and press Paste
  5. Do the same thing as step 4 & 5 but for your Pokemon Prism.sav file
    • Now your sav.dat file and pokeprism.sav file should be in the Pokemon Prism Save Converter folder and it should look like this.
  6. Open the helper application.exe file.
    • The Application should terminate itself when finished, this process should take a few seconds
  7. A new file named conv.dat should have been created in the Pokemon Prism Save Creator folder
  8. Right click on conv.dat press Copy, navigate to X:\SD\JKSV\Saves\Pokemon_Prism\Fresh and now right click and press Paste
  9. Delete the sav.dat file in the Fresh folder
  10. Rename conv.dat to sav.dat
    • If everything went to plan, there should only be 3 files in the Fresh folder, which are rsm1.dat rsm2.dat sav.dat

Section III - Importing the save

  1. Put the SD Card back into the 3ds and power it on
  2. Open up JKSM and press (A) on SD/CIA
  3. Hit (A) on Pokemon Prism
  4. Hit (A) on Save Data options
  5. Hit (A) on Import Save
  6. Hit (A) on Fresh
  7. Hit (A) to confirm the action
    • A message box saying "Complete!" should appear if succesful, if not repeat steps 3-7
