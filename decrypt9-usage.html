Decrypt9 Usage

Decrypt9 is outdated. If you dont have a good reason to use it, you should use GodMode9 instead

Creating a NAND backup

  1. Navigate to SysNAND Options
  2. Navigate to SysNAND Backup/Restore... and NAND Backup (min size)
    • This process takes some time.

Restoring a NAND backup

  1. Navigate to SysNAND Options
  2. Navigate to SysNAND Backup/Restore... and NAND Restore (keep hax)
    • This process takes some time.

Performing a CTRTransfer

  1. Make sure your CTRTransfer image is in /files9/
  2. Navigate to SysNAND Options
  3. Navigate to CTRNAND transfer... and Auto CTRNAND Transfer
    • This process takes some time.
