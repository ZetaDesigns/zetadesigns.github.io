Dumping with GodMode9

The dumped CIAs will be in /gm9out/

What you will need:

  • the latest release of GodMode9
  • enough space on your SD-Card

Dumping a cartridge game to CIA

  1. Hold start before powering on, to get into GodMode9
  2. Navigate to [C:] GAMECART
  3. Press (A) on titleID.trim.3ds
  4. Select NCSD image options... and Build CIA from file
    • This process takes some time
  5. Your dumped CIA will be in /gm9out/

Dumping an installed title to CIA

  1. Hold start before powering on, to get into GodMode9
  2. Navigate to [A:] SYSNAND SD
  3. Go to title and 000400000 for a full title, or 0004000e for update data
  4. Navigate to the folder with the last 8 digits of your titleID, and go to content
    • If you don't know what your titleID is, you can find a list here
  5. Press (A) on the .tmd file
  6. Select TMD file options... and Build CIA (standard)
    • This process takes some time
  7. Your dumped CIA will be in /gm9out/
